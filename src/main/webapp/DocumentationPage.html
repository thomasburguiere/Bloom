Masks validation tool

The mask validation tool valid or not each occurencce are correctly included in raster file.

What's a raster file ?

In its simplest form, a raster consists of a matrix of cells (or pixels) organized into rows and columns (or a grid) where each cell contains a value representing information, such as temperature. Rasters are digital aerial photographs, imagery from satellites, digital pictures, or even scanned maps. The dimension of the cells can be as large or as small as needed to represent the surface conveyed by the raster dataset and the features within the surface. For more information, go to ArcGIS.

Where is the problem ?

Tools for raster treatment like (?????) use informations contained in cells. Sometimes, cells have informations for land cells and not for water cells. For the last, these programs don't suport these occurences. Often, a filtering is necessary before.

Our mask validation tool do this checking to set appart all wrong occurences. To use this validation, you have to insert one or more raster files. VERIFIER FORMAT PRIS EN CHARGE. You can found raster files here : WorldClim
