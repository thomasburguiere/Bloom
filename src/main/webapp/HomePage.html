What Is BLOOM Doing?

Mapping to DarwinCore standard


Thanks to a friendly interface, so easy to use, you could easily convert all your csv files to the DarwinCore standard. Users will be able to select a column of their data set and choose the corresponding column in DarwinCore.

Find the correct scientific name


The tool integrates a reconciliation service API, which returns a list of potential entities matching the criteria from text. Many addresses exist to resolve syntaxic errors (some could be found here :
dev.gbif.org/wiki/display/POR/Webservice+API
biodiversitycatalogue.org/services/84).

A raster problem ?


Raster files that contain a lot of informations on temperature, altitude, etc in a matrix cells (lines and columns). Commonly, there are cells with informations on a land but missing on water cells. This lack of informations made errors during treatment process of these raster files.
